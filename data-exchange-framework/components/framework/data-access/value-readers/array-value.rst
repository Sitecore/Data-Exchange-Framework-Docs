Array Value
==========================================

This *value reader* is used to read a value from an array.

+-----------------+-----------------------------------------------------------+
| Template name   | **Array Value Reader**                                    |
+-----------------+-----------------------------------------------------------+
| Base template   | :doc:`base-value-reader`                                  |
+-----------------+-----------------------------------------------------------+

+-----------------------------------------------+-----------------------------------------------------------+
| Field                                         | Description                                               |
+===============================================+===========================================================+
| ``Position``                                  | The position in the array whose value is read (0-based).  |
+-----------------------------------------------+-----------------------------------------------------------+
