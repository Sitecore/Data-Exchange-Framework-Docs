Value Writers
=======================

As its name suggests, a *value writer* is able to write a value to 
an object. The object to which the value is written is called the
*target object*.

.. note::
    These templates are defined in the following location:

    **Templates > Data Exchange > Framework > Data Access > Value Writers**
