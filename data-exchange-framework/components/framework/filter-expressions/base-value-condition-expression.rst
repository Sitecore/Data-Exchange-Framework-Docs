Base Value Condition Expression
==========================================

This template is the base template for all *filter expressions*.

.. include:: ../../../../common/base-template-always-inherit-notice.txt

+-----------------+-----------------------------------------------------------+
| Template name   | **Base Value Condition Expression**                       |
+-----------------+-----------------------------------------------------------+
| Base template   | none                                                      |
+-----------------+-----------------------------------------------------------+

+--------------------------+--------------------------------------------------------------------------------+
| Field                    | Description                                                                    |
+==========================+================================================================================+
| ``Value Accessor``       | | The value accessor that is used to read the value that is used               |
|                          | | to build the filter expression that is compatible with the                   |
|                          | | provider that includes the template.                                         |
+--------------------------+--------------------------------------------------------------------------------+
