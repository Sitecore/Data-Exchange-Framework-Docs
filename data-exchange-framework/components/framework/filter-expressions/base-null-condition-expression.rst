Base Null Condition Expression
==========================================

This template is the base template for *filter expressions* that 
compares a value to *null*.

.. include:: ../../../../common/base-template-always-inherit-notice.txt

+-----------------+-----------------------------------------------------------+
| Template name   | **Base Null Condition Expression**                        |
+-----------------+-----------------------------------------------------------+
| Base template   | :doc:`base-value-condition-expression`                    |
+-----------------+-----------------------------------------------------------+

+-----------------+---------------------------------------------------------------------+
| Field           | Description                                                         |
+=================+=====================================================================+
| ``Operator``    | | Determines whether the condition checks to see if the value       |
|                 | | is or is not null.                                                |
+-----------------+---------------------------------------------------------------------+
