Base Numeric Condition Expression
==========================================

This template is the base template for *filter expressions* that 
compares a value to number.

.. include:: ../../../../common/base-template-always-inherit-notice.txt

+-----------------+-----------------------------------------------------------+
| Template name   | **Base Numeric Condition Expression**                     |
+-----------------+-----------------------------------------------------------+
| Base template   | :doc:`base-value-condition-expression`                    |
+-----------------+-----------------------------------------------------------+

+---------------------------+---------------------------------------------------------------------+
| Field                     | Description                                                         |
+===========================+=====================================================================+
| ``Condition Operator``    | Determines how to compare the value to the specified number.        |
+---------------------------+---------------------------------------------------------------------+
| ``Value``                 | The number that the value is compared to.                           |
+---------------------------+---------------------------------------------------------------------+
