Base String Condition Expression
==========================================

This template is the base template for *filter expressions* that 
compares a value to string.

.. include:: ../../../../common/base-template-always-inherit-notice.txt

+-----------------+-----------------------------------------------------------+
| Template name   | **Base String Condition Expression**                      |
+-----------------+-----------------------------------------------------------+
| Base template   | :doc:`base-value-condition-expression`                    |
+-----------------+-----------------------------------------------------------+

+---------------------------+---------------------------------------------------------------------+
| Field                     | Description                                                         |
+===========================+=====================================================================+
| ``Condition Operator``    | Determines how to compare the value to the specified string.        |
+---------------------------+---------------------------------------------------------------------+
| ``Value``                 | The string that the value is compared to.                           |
+---------------------------+---------------------------------------------------------------------+
