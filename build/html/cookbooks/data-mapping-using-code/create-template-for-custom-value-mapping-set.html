Create Template For Custom Value Mapping Set¶

In order to be able to configure the data synchronization process to use custom value mapping set, a template is needed.

In this topic:

  • Create Template
  • Set Icon
  • Add Standard Values
  • Remove Insert Option For Value Mapping

Create Template¶

  1. In Sitecore, open Template Manager.
  2. Add a new template.
  1. Enter the following values:
Name Custom Value Mapping Set
Base template Templates > Data Exchange > Framework > Data Access > Mapping > Value Mapping Set
  1. Click Next.
  1. Select Templates > Data Exchange > Framework > Data Access > Mapping.
  1. Click Next.
  2. Click Close.

Set Icon¶

  1. Select the new template.
  1. Click the Content tab.
  1. Click the icon.
  1. Enter Office/32x32/vector_circle.png
  1. Click OK.

Add Standard Values¶

  1. Click the Builder tab.
  1. In the ribbon, click Builder OPTIONS.
  1. Click Standard values.
  1. Select the new item.

Remove Insert Option For Value Mapping¶

Since all of your value mapping is done in code, there is no reason to allow users to add value mapping items under the custom value mapping set.

  1. In the ribbon, click CONFIGURE.
  1. Click Assign.
  1. In the list on the right, select Value Mapping.
  1. Click the left arrow.
  1. Click OK.
