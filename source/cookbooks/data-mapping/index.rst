Data Mapping
===================================================
An essential aspect of data synchronization processes
is the mapping of data from one system to another.
This section provides examples for developers on 
how to configure and customize the data mapping
process.

.. note::

    Finished code and installation packages are available in 
    `GitHub <https://github.com/Sitecore/Sitecore.DataExchange.Examples/>`_.
